Introduction

This guide will help you add sqlcommenter to your Django applications running locally.

Please see the reference for the fields added in the SQL comments google-cloud-sqlcommenter.Fields

Addition to your code

Firstly, please install google-cloud-sqlcommenter.

For any Django deployment, we can just edit our settings.py file and update the MIDDLEWARE section as per:

MIDDLEWARE = [
  'google.cloud.sqlcommenter.django.middleware.SqlCommenter',
  ...
]
If any middleware execute database queries (that you'd like commented by SqlCommenter), those middleware MUST appear after 'google.cloud.sqlcommenter.django.middleware.SqlCommenter'
